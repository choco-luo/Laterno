# 🐳Laterno
靜態部落格，使用[Astro](https://astro.build/)製作。

「Laterno」是一個以海洋為靈感的部落格平台，提供輕盈且可擴展的創作體驗。使用者可以自由發表文章、自定 Tag、追蹤喜愛的海洋生物，並透過個人後台與卡片建立專屬的數位身份，還有互動島嶼與友情連結功能，讓內容、興趣與社群自然匯流，開放給所有有興趣的人使用。
