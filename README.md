# 🐳Laterno
靜態部落格，使用[Astro](https://astro.build/)製作。

以海洋為主題的部落格網站，開放給所有有興趣的人使用。

- [ ] 自定義文章Tag

- [ ] 類似Abyssrium的小型互動島嶼
